# Fly With Maths (Prototype)

Vertical sci‑fi military shooter in HTML5 Canvas.

## Features
- Player ship with forward lasers
- Fighters, bombers, hybrid enemies
- Power‑ups that float downward and change stats
- Health, score, HUD
- Jammer Mode (Ctrl + 7 disables enemy shooting for 30 seconds)
- Weather: clear, rain, fog, storm
- Level zones: forest, river, city, base
- Boss fight with health bar

## Run
Open `index.html` in a browser.
